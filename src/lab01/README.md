### Задание 6
``` python
members = int(input('in_1: '))
ochno = 0
zaochno = 0
for i in range(members):
    member = input(f'in_{i+2}: ')
    if 'True' in member:
        ochno +=1
    else:
        zaochno+=1
print(f'out: {ochno} {zaochno}')
```

### Задание 7
``` python
vxod = input('in: ')
word = ''
index1 = 0 
index2 = 0

for i in range(len(vxod)):
    if vxod[i].isupper():
        index1 = i 
        break
    else:
        continue 
for i in range(len(vxod)):
    if vxod[i] in '0123456789':
        index2 = i+1
        break
    else:
        continue
shag = index2 - index1
for i in range(index1,len(vxod),shag):
    word+=vxod[i]
print(f'out: {word}')
```
