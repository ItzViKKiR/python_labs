## Объект

Более детально: [LAB01](/Year1_Sem2/labs/lab01)

## Коллекция
District — контейнер объектов Property.

Возможности:
- добавление и удаление объектов
- поиск по владельцу
- итерация
- индексация
- сортировка
- фильтрация

## Реализовано
- add / remove / remove_at
- find_by_owner
- __len__, __iter__, __getitem__
- sort / sort_by_price
- get_available
- get_with_debt
