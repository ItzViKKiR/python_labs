# ЛР-4 — Интерфейсы и абстрактные классы

## Предметная область
Недвижимость

## Объекты
Система управления недвижимостью:
- объекты аренды
- объекты с ипотекой

## Интерфейсы

### Printable
Отвечает за вывод информации об объекте

Метод:
- to_string()

### Payable
Отвечает за расчёт платежей

Метод:
- calculate_payment()

## Классы

### Property
Базовый класс недвижимости

Более детально: [ТЫЫЫЫЫК](/Year1_Sem2/labs/lab01)

### RentalProperty и MortgageProperty
Недвижимость для аренды/с ипотекой

Более детально: [ТЫЫЫЫЫК](/Year1_Sem2/labs/lab03)


## Реализация

- Использованы абстрактные классы (ABC)
- Интерфейсы задают контракт поведения
- Реализован полиморфизм через интерфейсы
- Добавлены универсальные функции:
  - print_all()
  - calculate_all()
